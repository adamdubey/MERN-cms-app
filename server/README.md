# mern-cms-server

## Quick Start

_via Localhost (development):_
```sh
# install dependencies
npm install

# run server-app 
npm run start
```

_via Docker:_
```sh
# build container image
docker build -t cms-server .

# run container image
docker run -d -p 8000:8000 cms-server:latest
```
