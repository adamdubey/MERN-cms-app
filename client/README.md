# mern-cms-client

## Quick Start

_via Localhost (development):_
```sh
# install dependencies
npm install

# run client-app 
npm run dev

# in browser, visit http://localhost:3000
```

_via Docker:_
```sh
# build container image
docker build -t cms-client .

# run container image
docker run -d -p 3000:3000 cms-client:latest

# in browser, visit http://localhost:3000
```
